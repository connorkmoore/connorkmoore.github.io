## Overview

Built a virtualized home lab environment using Proxmox and developed a pan-tilt AI object detection system using Raspberry Pi and computer vision libraries.

## Technologies Used

- Computer Vision
- Proxmox
- Raspberry Pi

## Project Type

Personal Project

## Technical Details

This project combines virtualization technology with AI/ML capabilities to create a flexible prototyping environment. The pan-tilt object detection system demonstrates practical application of computer vision algorithms on embedded hardware.
